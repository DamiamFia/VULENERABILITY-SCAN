# Network Vulnerability Assessment

##  Overview
This project demonstrates a full vulnerability assessment on a simulated network using industry-standard tools.

##  Objectives
- Identify open ports and services
- Detect vulnerabilities
- Assess risk levels
- Recommend mitigation strategies

## Tools Used
- Nmap
- Nessus Essentials
- Metasploitable

##  Methodology
1. Network discovery
2. Port scanning
3. Service enumeration
4. Vulnerability scanning
5. Host scanning 
6. Risk analysis

## Key Findings
| Vulnerability | Severity | Description |
|--------------|---------|------------|
| Open SSH Port | Medium | Exposed SSH service |
|  | High | Vulnerable to exploits |

## Recommendations
- Disable unused ports
- Patch outdated software
- Implement firewall rules
